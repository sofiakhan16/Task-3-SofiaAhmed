# E-Commerce SQL Data Analysis Project

## 📌 Project Overview

This repository contains the SQL Data Analysis phase of the E-Commerce Analytics project, completed as part of the DecodeLabs Data Analytics Internship. Using a cleaned E-Commerce Orders dataset, the project focuses on extracting meaningful business insights through structured SQL queries.

The analysis demonstrates core SQL concepts including data retrieval, filtering, sorting, aggregation, and grouping to answer business-oriented questions related to sales performance, customer activity, product trends, payment preferences, and revenue generation.

## 🛠️ Tools & Technologies

* **SQL**
* **SQLite**
* **E-Commerce Orders Dataset**

## 🔍 SQL Concepts Applied

* `SELECT` Statements
* `WHERE` Clause Filtering
* `ORDER BY` Sorting
* `GROUP BY` Aggregation
* Aggregate Functions:

  * `COUNT()`
  * `SUM()`
  * `AVG()`
* `HAVING` Clause for Group Filtering
* `DISTINCT` Value Extraction

## 📊 Analysis Highlights

* Retrieved and explored order-level transaction data.
* Analyzed customer and product performance.
* Calculated total revenue, average order value, and items sold.
* Evaluated payment methods and referral sources.
* Identified top-performing products by revenue.
* Generated grouped business summaries using SQL aggregations.

## 🚀 Key Outcome

This project demonstrates the practical application of SQL for transforming raw transactional data into actionable business insights. It highlights the ability to query, filter, aggregate, and analyze data efficiently using industry-standard SQL techniques, forming a strong foundation for data analytics and business intelligence workflows.
